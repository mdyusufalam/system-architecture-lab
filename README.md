# SystemArchitectLab

A comprehensive learning repository for system architecture patterns and resilience strategies in .NET.

## 📚 Projects

| SL | Project Name | Description | Documentation |
|:--:|:-------------|:-----------|:--------------|
| 1 | Circuit Breaker Lab | Demonstrates Circuit Breaker pattern using Polly resilience library in .NET 10. Learn failure handling, state management, and graceful degradation. | [README.md](./src/CircuitBreakerLab/README.md) |

## 🛠️ Prerequisites

- .NET 10 SDK or higher
- Visual Studio, VS Code, or any .NET-compatible IDE
- Basic understanding of async/await in C#

## 🚀 Quick Start

```bash
cd src/CircuitBreakerLab
dotnet run
```

Visit `http://localhost:5000/weather` to test the API.
